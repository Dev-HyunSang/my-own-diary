# my own diary
많은 분들께서 다이어리를 쓰시는 모습을 보고 "다이어리만 쓸 수 있는 플랫폼이 있으면 어떨까?"라는 생각이 들어서 개발하게 되었습니다.    
"**나의 생각**"을 담는 서비스인 만큼 보안을 신경 써서 개발하고 있습니다.
